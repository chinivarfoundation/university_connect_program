## Project Overview
Chinivar Foundation aims to build a single, scalable digital platform to operate as a federated NGO with multiple chapters and programs. The system is designed to ensure financial transparency, compliance, donor trust, and measurable impact, with the long-term vision of scaling to 10+ chapters, 10,000+ donors, 5,000+ students, and running a full-fledged school.
